# Marine Interview App

해기사 면접 기출문제 주관식 학습 앱입니다.

## 파일 구성
- `index.html`: GitHub Pages 시작 파일
- `assets/app.js`: 앱 기능
- `assets/style.css`: 디자인
- `data/questions.js`: 문항 데이터

## 문항 수
- 주관식 문항: 2225개
- 원문 보존 블록: 166개

## GitHub Pages
Repository 루트에 이 파일들을 넣고 `Settings > Pages > Deploy from a branch > main > /(root)`로 설정하세요.
